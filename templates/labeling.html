Unser Annotationstool bietet eine ansprechende Benutzeroberfläche, in der die Tweets, die für die Analyse ausgewählt wurden, übersichtlich präsentiert werden. Der Annotator verfügt über Schaltflächen, welche es ermöglichen das Sentiment eines Tweets als positiv, negativ oder neutral zu kennzeichnen. Zusätzlich bietet die Benutzeroberfläche die Möglichkeit, Tweets, die für die Analyse irrelevant sind, auszuschließen. Nach einer gewissen Anzahl von Annotationen, kann das entsprechende Machine Learning Modell trainiert werden und dem Annotator Unterstützung leisten. Beim Drücken auf "Intelligent sortieren" werden die Tweets nach einer Kombination von Faktoren geordnet, die eine möglichst optimale Performance des Machine Learning Modells erlauben.
