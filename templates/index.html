Tweet Sentiment Analyse mithilfe von Active Learning

Um das Sentiment von Tweets zu analysieren, geben Sie in das Feld "Suchbegriff" Schlüsselwörter oder Phrasen ein, die in Bezug auf das stehen, worüber Sie die Analyse durchführen möchten. Zusätzlich müssen Sie einen Zeitraum angeben, auf den sich die Tweets beziehen sollen, eine Sprache auswählen und die Anzahl der Tweets festlegen. Klicken Sie auf die Schaltfläche "Erstellen", um die Ergebnisse im Dropdown Menü, der bestehenden Datensätze anzuzeigen.

Neuen Datensatz erstellen

Wählen sie um fortzufahren dann einen neu erstellten Datzensatz oder einen bereits bestehenden Datensatz aus dem Dropdown Menü aus um diesen zu löschen oder herunterzuladen. Wenn Sie die Tweets annotieren möchten, wählen Sie diesen aus und Sie werden zum Annotationstool weitergeleitet.
